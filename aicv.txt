1. Fibonacci Series (ASP.NET C#)
Code:
protected void B1_Click(object sender, EventArgs e)
{
    int n=int.Parse(T1.Text),a=0,b=1,c;
    for(int i=0;i<n;i++){
        L1.Text+=a+" ";
        c=a+b; a=b; b=c;
    }
}

Explanation:
Generates Fibonacci numbers using a loop where each number is the sum of the previous two numbers.

2. Prime Number Check
Code:
int n=int.Parse(T1.Text),c=0;
for(int i=1;i<=n;i++) if(n%i==0) c++;
L1.Text=(c==2)?"Prime":"Not Prime";

Explanation:
Counts how many numbers divide n. If exactly two divisors exist (1 and n), the number is prime.

3. Factorial Program
Code:
int n=int.Parse(T1.Text),f=1;
for(int i=1;i<=n;i++) f*=i;
L1.Text="Factorial="+f;

Explanation:
Multiplies numbers from 1 to n to compute factorial.

4. Reverse Number and Sum of Digits
Code:
int n=int.Parse(T1.Text),rev=0,sum=0;
while(n>0){
 int d=n%10;
 rev=rev*10+d;
 sum+=d;
 n/=10;
}
L1.Text="Rev="+rev+" Sum="+sum;

Explanation:
Extracts digits using modulus and builds the reversed number while summing digits.

10. Disconnected Database Example
Code:
SqlDataAdapter da=new SqlDataAdapter("select * from student",con);
DataSet ds=new DataSet();
da.Fill(ds,"student");
DataRow r=ds.Tables["student"].NewRow();
r["name"]=txtName.Text;
r["age"]=txtAge.Text;
ds.Tables["student"].Rows.Add(r);
new SqlCommandBuilder(da);
da.Update(ds,"student");

Explanation:
Uses DataSet and SqlDataAdapter so the connection is not continuously open.
